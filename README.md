# Don't Starve Together Kanban Board

A thematic Kanban board webapp built with Next.js, featuring a Don't Starve Together inspired dark, atmospheric design.

## Features

- Drag and drop tasks between columns
- Add new tasks to any column
- Don't Starve Together themed UI

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- @hello-pangea/dnd (drag and drop)
